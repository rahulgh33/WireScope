# Contributing

Pull requests welcome. 

## Before submitting

- Run tests: `make test`
- Format code: `go fmt ./...`
- Lint: `golangci-lint run`

## What to contribute

Check [tasks.md](.kiro/specs/network-qoe-telemetry-platform/tasks.md) for planned work.

Useful areas:
- Bug fixes
- Performance improvements
- Additional probe metrics
- Storage backend adapters
- Deployment examples
- Documentation fixes

## Code style

- Keep it simple
- Avoid over-engineering
- Write tests for new features
- Document non-obvious behavior in code comments

## Issues

Open an issue before starting major work to discuss the approach.
